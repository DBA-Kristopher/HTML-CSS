Welcome to Overwatch

This is Overwatch fan site for helpful macros created by Kristopher Smith. These macros allow for advance tactics and combos with little to no user interaction. Many of these have been created with the purpose of eliminating mindless or annoying repetitive task Overwatch players normally experince.

Software Requirements

  • Auto Hot Key

    • You will need to download and install Autohotkey which is a free basic macro and scripting lanuage that allows users to program macros often for game related software.

  • Overwatch

    • Next you will need to have the overwatch client installed. This is the game the macros are intended to work in. This will have to be purchased from blizzard entertainment.
