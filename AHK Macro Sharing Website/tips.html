Tips

When creating overwatch macros it is important to not use any code that makes request from either the battlenet.exe or the overwatch.exe to gather data. This is mostly important because blizzard detection system will catch any users that mod game files or make request into game files. This is why using outside third party scripts such as AHK are so useful in creating a competitive advantage. The operate outside the actual game systems and are being used by the OS instead of the game application.

Editing your own macros

It is a very simple process once you get a shell of a macro working. To create your own as a first time user it is best to open the AHK files in a txt file and only replace code with something your would like to change For example using a bunny hop macro that uses the spacebar changing it to toggle on a specific button instead.
